# ERP

Este repositorio contiene una aplicación de tipo ERP desarrollada en Python con interfaz gráfica utilizando PyQt5.

El proyecto está estructurado de forma modular para separar la lógica de negocio, la gestión de datos y la interfaz de usuario.

## Estructura del proyecto

- **Main.py**: Archivo principal que inicia la aplicación y carga la interfaz.
- **Components/**: Contiene la lógica de negocio y funcionalidades del sistema.
- **Data/**: Gestión de datos, incluyendo carga y almacenamiento de información.
- **UI/**: Componentes de la interfaz gráfica de usuario.

## Tecnologías utilizadas

- Python 3
- PyQt5

## Instalación

Clona el repositorio:

```bash
git clone https://github.com/PacoMorales03/ERP.git
cd ERP
```

Instala las dependencias:

```bash
pip install -r requirements.txt
```

## Ejecución

```bash
python Main.py
```
